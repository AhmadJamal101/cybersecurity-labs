## General Notes

### Introduction to Cybersecurity

- Cybersecurity focuses on protecting systems, networks, and data.
- It includes both offensive and defensive security concepts.

### Offensive Security Intro

- Offensive security focuses on understanding how attackers think and operate.
- The goal is to identify weaknesses before real attackers can exploit them.

### Defensive Security Intro

- Defensive security focuses on protecting systems, monitoring activity, and responding to threats.
- It includes areas like SOC, SIEM, threat intelligence, and incident response.

### Careers in Cyber

- Cybersecurity has multiple career paths, including SOC Analyst, Penetration Tester, Security Engineer, Penetration Tester, and Digital Forensics.
